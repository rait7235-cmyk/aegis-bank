Aegis Bank – Landing Page

 Description:

  -Projet de création d’une landing page pour une néobanque éthique et transparente.
   Le site a été conçu en mobile-first avec une version mobile sans scroll.



 Objectifs:

- Créer une interface moderne et accessible
- Respecter une architecture multi-pages
- Appliquer les principes UX/UI
- Développer en HTML/CSS pur



 Fonctionnalités:

- Design responsive (mobile-first)
- Landing page mobile sans scroll
- Navigation entre plusieurs pages
- Animations CSS (micro-interactions)
- Accessibilité (WCAG)



 Technologies utilisées:

- HTML
- CSS3
